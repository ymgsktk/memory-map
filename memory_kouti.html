思い出マップの使い方

1.「思い出マップ」は訪れたことのある都道府県の思い出を記録するアプリです

2. 訪れたことのある都道府県をクリックしてその都道府県に関する思い出を記録していきます

3. 日本地図で灰色は未訪問，緑色は訪問済みを表しています

4. 日本地図を全て埋めて，コンプリートを目指そう！

思い出マップの記録方法

1. 　地図の都道府県をクリックすると，都道府県ごとに思い出の写真を記録しておくことができます

2. 　画面下部の『思い出を書き留める』より記録したい写真を選択し，日付とメモを設定します

3. 　紙飛行機アイコンをクリックすることでギャラリーに記録することができます

4. 　ギャラリーの画像部分をクリックすると記録を削除できます

5. 　『この内容で記録』ボタンをクリックすることで，その都道府県に訪れたことを記録します
